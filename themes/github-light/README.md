# GitHub Light Theme

This built-in theme uses `github-markdown-light.css` from
[`sindresorhus/github-markdown-css`](https://github.com/sindresorhus/github-markdown-css),
version 5.9.0.

The upstream stylesheet is licensed under the MIT License. The original license
is included as `LICENSE.github-markdown-css`.

MWRender embeds the stylesheet at build time so the default theme remains
available after installation and does not depend on the process working
directory.
